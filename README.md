# box-pop-demo

BOX POP《盲盒异想消》｜Cocos Creator 3D 物理点消手机试玩版。

本版本包含乳白漫射背光舱、四角暖白玻璃灯、轻量接触阴影、点击连锁边缘高亮，以及六面动态表面的功能盲盒。

## 本次更新

- 新增兔兔主题主界面与“开始游戏”流程。
- 游戏左下角加入兔兔伙伴和技能槽；一次消除 7 个以上固定生成炸弹并为兔兔充能。
- 舱体外圈加入低干扰压力仪表；连续消除升温，满值进入 6 秒“火热出货”状态，冻结倒计时并提供双倍得分。
- 炸弹冲击动画改成两段式慢抛飞，保留力度的同时提升可读性与观赏性。
- 右上角设置按钮可全屏暂停游戏、继续游戏或选择 1–50 关。
