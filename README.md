# box-pop-demo

BOX POP《盲盒异想消》｜Cocos Creator 3D 物理点消手机试玩版。

本版本包含乳白漫射背光舱、四角暖白玻璃灯、轻量接触阴影、点击连锁边缘高亮，以及六面动态表面的功能盲盒。

## 本次更新

- 新增兔兔主题主界面与“开始游戏”流程。
- 游戏左下角加入兔兔伙伴和技能槽；一次消除 7 个以上固定生成炸弹并为兔兔充能。
- 舱体外圈加入低干扰压力仪表；连续消除升温，满值进入 6 秒“火热出货”状态，冻结倒计时并提供双倍得分。
- 炸弹冲击动画改成两段式慢抛飞，保留力度的同时提升可读性与观赏性。
- 右上角设置按钮可全屏暂停游戏、继续游戏或选择 1–50 关。

在线试玩：https://bisoubisou0905.github.io/box-pop-demo/

## 2026-07-24 POLAR 标准宽长方体

- 新增深海军蓝 `POLAR` 雪地犬系列，使用独立 3×2 六面印刷图，正面、背面、左右侧与顶底都有对应包装内容。
- 新增第三种程序化盲盒模型 `TALL_BOX`：比现有方盒稍宽、明显更高，并配套独立质量、长方体碰撞代理、阴影与选中高亮。
- `POLAR` 只加入点消物理模式的轮换池；传统三消继续只使用方盒与球盒，避免长方体破坏网格可读性。
- 手机 Web 构建和两种玩法均已实机尺寸检查，控制台无资源或脚本错误。

## 2026-07-22 交互与性能打磨

- 压力仪表改为贴合玻璃外圈的 16 段低对比度刻度，不再遮挡舱体。
- 兔兔与技能槽重构为机器左下角的一体式角色底座，并加入待机、点头、欢呼和强摇反应。
- 抬手时重新计算实时连通组；已经滚出连接范围的盲盒不会被误消除。
- 支持可选手机运动传感器；只有强烈摇晃才产生极轻微镜头反馈。
- 物理、阴影、功能盒表面动画和 UI 改用分频更新，保留材质与光照并降低持续发热。

## 2026-07-23 通关爆发与体感

- 通关后先进入约 3 秒的 `BONUS RUSH`，自动完成当前真实三连并快速累计奖励分。
- 庆祝阶段会从隐藏补货口高速送入一批盲盒，再用三段炸弹连锁把舱内盲盒击飞，最后才显示结算卡。
- 结算卡改为动画结束后再进入渲染树，避免提前遮挡通关演出。
- 微信加速度计与网页 DeviceMotion 都会在玩家强烈摇晃时给盲盒施加极小物理冲量；体感幅度远低于重排和踢机器。

## 2026-07-23 传统三消成长系统

- 传统三消的方形物理舱会随章节逐步扩大，容量由约 30 件提升到 43 件，补货通道由 6 条增加到 8 条。
- 三消模式接入独立的标准、奖励和卡关数值：奖励关减少颜色压力并增加步数，卡关提高目标和颜色熵并减少步数。
- 每个三消关卡使用固定种子生成开局；盲盒落稳后会先校准所有预存直线，确保玩家操作前不会自动连消。
- 任意方向近似直线仍可触发三消，但连锁只会从玩家完成的有效交换开始。
